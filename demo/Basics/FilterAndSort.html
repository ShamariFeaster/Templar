Sorting

`Model.sort(string attributeName)` will not update the view automatically. You need to call `Model.update(string attributeName)` to see the sort reflected in the view.

Primative Arrays

Using Templar, performing complex sorting of`DMA`s is easy. The markup for the list of numbers below is: ```
```

The code:

``` $('.sort').click(function(e){ Example.sort('numbers'); Example.update('numbers'); }); ```


Arrays of objects

To sort an array of objects, you can specify the object property to order by. `Model.sort()` is chainable with `orderBy(string propertyName)`.

The markup:

```

Live example:

{{}}
{{Kingdom}}{{Class}}{{Species}}
```

The code:

``` $('.sortableColumn').click(function(e){ var colName = e.currentTarget.getAttribute('x-colName'), Taxonomy = Tempar.getModel('Taxonomy'); Taxonomy.sort('data').orderBy(colName); Taxonomy.update('data'); }); ```
