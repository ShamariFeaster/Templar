Using different types of lists

Notice the difference between the SimpleSelect and ComplexSelect `DMA`a.

Simple Select: Templar will use each element in the list as both the name and value
Complex Select: You can specify different names and values by binding to a list of objects that have name and value properties

The `current_selection` property

Once a list is bound to a select, it gets a "current_selection" property that can be used to get/set the value of the currently selected option.

Selecting an option using the `current_selection` property

To set a select's current selection, assign `current_selection` to a value that is in the list. In the case of a complex list, Templar will recognize both valid text and value properties

Select simple select's `current_selection` by typing a valid value here

  1. Type in '3' and notice the simple select changes to '3'
  2. Type in '2' and notice the simple select changes to '2'
